# Privacy Policy — ZiTan Chinese Popup Dictionary

**Last updated:** March 18, 2026

## Summary

ZiTan Chinese Popup Dictionary does not collect, transmit, or sell any user data. Everything runs locally in your browser.

## Data Storage

The extension stores the following data **locally on your device** using Chrome's storage API and IndexedDB:

- **User preferences** — theme, font size, tone colors, character display mode, blocked domains, and other settings you configure in the options page.
- **Dictionary data** — CC-CEDICT and Dong Chinese dictionary entries are cached in your browser's IndexedDB for fast lookups. This data is bundled with the extension and never fetched from external servers.

## Data Collection

This extension does **not**:

- Collect any personal information
- Track your browsing activity
- Send any data to external servers
- Use analytics or telemetry
- Use cookies
- Share or sell any data to third parties

## Network Requests

This extension makes **zero network requests**. All dictionary data is bundled locally within the extension. No external APIs are called.

## Permissions

- **Host permissions (all URLs)** — required to display dictionary popups on any webpage. No page content is read, stored, or transmitted.
- **Storage** — used to save your preferences locally.
- **Scripting** — used to inject the popup script into webpages.
- **ActiveTab** — used to toggle the extension on the current tab.
- **ContextMenus** — used to add a toggle option to the extension icon menu.
